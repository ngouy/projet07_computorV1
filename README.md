# projet07_computorV1
little reducer &amp;&amp; calculator of polynomials. All Libs are included.
Natural form supported. (dont forget quotes)

e.g:
  - "1 * X^0 + 2 * X^1 + 4 * X^2 = 0 * X^0 + 4 * X^1 + 3 * X^2"
  - "3x +2 -4x^5 = 3x^2 -1 +3* x^4"
